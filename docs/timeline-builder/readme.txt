# Harpoon Timeline Builder

A lightweight, GSAP-free timeline builder for Shorthand stories.

## Features
- Native horizontal scrolling (mobile & desktop)
- Inherits story typography
- Image + credit support
- Exportable HTML & CSS
- No editor CSS leakage

## Usage
1. Build your timeline
2. Copy HTML into a Shorthand embed
3. Copy CSS into story custom CSS

## Status
Stable (v1.5.2)
